## 数组

```
数组就是一个集合，里面存放了相同类型的数据元素
（1）数组中的每个数据元素都是相同的数据类型
（2）数组是由连续的内存位置组成的
```

### 一维数组

```
一维数组定义的三种方式:
（1） 数据类型 数组名[数组长度]；
（2） 数据类型 数组名[数组长度]={值1,值2...}
（3） 数据类型 数组名[]={值1,值2...}

一维数组数组名
作用：
（1）可以统计整个数组再内存中的长度 sizeof(arr)
（2）可以获取数组在内存中的首地址 cout<<arr<<endl;//此时输出的是数组的首地址而非其中的元素，不需要&进行取址操作
```

### 二维数组
```
二维数组定义的四种方式：
（1）数据类型 数组名[行数][列数]
（2）数据类型 数组名[行数][列数]={{数据1，数据2},{数据3，数据4}}
（3）数据类型 数组名[行数][列数]={数据1，数据2,数据3，数据4}
（4）数据类型 数组名[][列数]={数据1，数据2，数据3，数据4}

二维数组组名
作用：
（1）查看二维数组所占内存空间
（2）获取二维数组首地址
```
